Romans Cheatsheet for HTML / CSS

HTML Tags

Tag Name Beschreibung
<h1></h1> Überschrift h1-6 Überschriften in versch. Größen
<p></p> Paragraphen Textcontainer
<img src="" alt=""> Bildlink in src="" link zum Bild in alt="" Bildbeschreibung
<a href=""></a> Link Link auf andere Seite oder # auf der selben Seite
<link href="" rel=""> Link um zum Beispiel ein stylesheet in html zu implimentieren (styles.css)
<div></div> Container Container mehrer HTML Zeilen
<article></article> Text Container Container mehrer Text Bausteine
<strong></strong> Bold Zum Bold schreiben einzelner Wörter in einem Paragraphen
<em></em> Italic Zum kursiv schreiben einzelner Wörter in einem Paragraphen
<span></span> Span zum gruppieren und bearbeiten von inline Inhalten
<br> Linebreak produziert einen Zeilenumbruch in (Fließ-)Text
<aside></aside> aside repräsentiert eine Sektion einer Seite, welche aus Inhalt besteht der nur indirekt zum restlichen Inhalt gehört, also parallel zum Kontext betrachtet werden kann.
<cite></cite> Quellenangabe Angabe von Quellen die nicht direkt Lesbar sind, aber in den Metadaten gefunden werden können
<hr> horizontal rule wird verwendet, um eine horizontale Linie einzufügen
<nav></nav> Navigationsleiste zeichent einen Seitenbereich aus, der dazu bestimmt ist Navigationslinks bereitzustellen
<q></q> Quote (=Zitat) indiziert, dass es sich bei dem eingeschlossenen Text um ein kurzes Zitat handelt (kursiv geschrieben)
<ol></ol> orderd list geordnete Liste (nach zahlen geordnet)
<ul></ul> unorderd list ungeordnete Liste (nach punkten geordnet)
<li></li> Listenpunkt um Listenpunkte hinzuzufügen (child of ul/ol)

CSS Tags

Tag Name Beschreibung
color: *; Foregroundcolor Zum färben von Inhalten im Vordergrund (=Text)
background-color: *; Backgroundcolor zum färben vom Hintergrund
font-size: *px; Schriftgröße ändert die Schriftgrößen (*px, *em)
font-family: "*", *, *; Schriftart Ändert die Schriftart, mehrer Angeben, bei Fehler wird Text sonst nicht dagestellt
text-align: *; Textausrichtung Richtet den Text aus (left, center, right)
align-items / justify-content: *; Ausrichtung von Flexelementen Ausrichtung innerhalb eines Container horizontal / vertical
font-weight: *; Text formatierung Zum formatieren in den Fettdruck (bold, 100-900,...)
border: *; Rahmen Umrandung von HTML Elementen (solid, thick, pointed ,... / color / *px )
border-radius: *px/%; Rahmenecken um Rahmen mit abgerundeten Ecken zu versehen
(50% = rund)
background-image: url("*") Hintergrundbild Hintergrundbild in einem Container (Siehe Header)
